# Project setup
Start by cloning this repository

```
# HTTPS
$ git clone https://github.com/yogasab/techincal-test-fast-8.git
```

then
```sh
# cd into project root and install dependencies
$ npm i (dependencies)
# to use pgsql
$ npm i pg pg-hstore
# start the api using nodemon
$ nodemon server.js or npm run dev
```

### Run
```
nodemon server.js or npm run dev
```
